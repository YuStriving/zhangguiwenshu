## 🎨 技术选型理由

### 1. FastAPI 选型理由
- **高性能**: 基于 Starlette 和 Pydantic
- **异步支持**: 完整的 async/await 支持
- **自动文档**: 自动生成 OpenAPI 文档
- **类型安全**: 基于 Python 类型提示

### 2. LangGraph 选型理由
- **工作流管理**: 专门为 Agent 工作流设计
- **状态管理**: 内置的状态管理机制
- **可视化**: 支持工作流可视化
- **扩展性**: 易于扩展新的处理节点

### 3. Qdrant 选型理由
- **高性能**: 专门为向量搜索优化
- **易用性**: 简单的 API 接口
- **Docker 支持**: 容器化部署简单
- **社区活跃**: 活跃的开发和社区支持

### 4. 依赖注入设计理由
- **解耦**: 服务间解耦，提高可测试性
- **可维护性**: 清晰的依赖关系
- **灵活性**: 易于替换实现
- **生命周期管理**: 统一的服务管理
